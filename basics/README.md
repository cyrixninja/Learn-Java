## For Loop Java
```

     System.out.println("Hello World :" + i);
}
```

## Switch Case Java

```
    switch (operator) {
        case '+':
        System.out.println("The Answer is : "+(x+y));
            break;  
        case '*':
        System.out.println("The Answer is : "+(x*y));
            break;  
        case '/':
        System.out.println("The Answer is : "+(x/y));
            break;  
        case '-':
        System.out.println("The Answer is : "+(x-y));
            break;  
        default:
        System.out.println("Invalid Values !");
            break;
    }
```
